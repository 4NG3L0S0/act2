10 Etiquetas en HTML3

<html>

La etiqueta <html> es el contenedor principal de todo el documento HTML. Toda la estructura de la página va dentro de esta etiqueta.

        <html>
            ... contenido de la página ...
        </html>

<head>

La etiqueta <head> contiene metadatos sobre el documento como el título, enlaces a hojas de estilo, scripts, etc. No se muestra en la página.

        <head>
            <meta charset="UTF-8">
            <title>Título de la página</title>
        </head>

<title>

La etiqueta <title> define el título del documento, que aparece en la pestaña del navegador.

        <title>Mi Página Web</title>

<body>

La etiqueta <body> es el contenedor principal del contenido que se muestra en la página web.

        <body>
            <h1>Bienvenidos a mi sitio web</h1>
            <p>Este es un párrafo de ejemplo.</p>
        </body>

<h1> a <h6>

Las etiquetas de <h1> a <h6> son usadas para definir encabezados, siendo <h1> el más importante y <h6> el menos importante.

        <h1>Este es un Encabezado 1</h1>
        <h2>Este es un Encabezado 2</h2>

<p>

La etiqueta <p> se usa para definir un párrafo de texto.

        <p>Este es un párrafo en HTML3.</p>

<a>

La etiqueta <a> crea un enlace o hipervínculo a otra página o recurso.

        <a href="https://www.ejemplo.com">Visita Ejemplo</a>

<img>

La etiqueta <img> se usa para insertar imágenes en la página. Es una etiqueta vacía, es decir, no tiene una etiqueta de cierre.

        <img src="imagen.jpg" alt="Descripción de la imagen">

<ul> y <li>

La etiqueta <ul> define una lista desordenada y <li> define cada elemento de la lista.

        <ul>
            <li>Elemento 1</li>
            <li>Elemento 2</li>
        </ul>

<table>

La etiqueta <table> se usa para crear tablas en HTML. Las etiquetas relacionadas son <tr> (filas), <th> (encabezados) y <td> (celdas).

        <table>
            <tr>
                <th>Nombre</th>
                <th>Edad</th>
            </tr>
            <tr>
                <td>Juan</td>
                <td>30</td>
            </tr>
        </table>
