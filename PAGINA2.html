15 Etiquetas Adicionales en HTML3

<div>

La etiqueta <div> es un contenedor genérico que se utiliza para agrupar contenido y aplicar estilos o scripts.

        <div>
            <p>Contenido dentro de un div.</p>
        </div>

<span>

La etiqueta <span> es un contenedor en línea utilizado para aplicar estilos a una parte del texto.

        <p>Este es un <span style="color:red;">texto rojo</span>.</p>

<strong>

La etiqueta <strong> se usa para enfatizar texto con importancia, lo que generalmente lo hace aparecer en negrita.

        <p>Este es un <strong>texto importante</strong>.</p>

<em>

La etiqueta <em> se utiliza para enfatizar texto, lo que lo hace aparecer en cursiva.

        <p>Este es un <em>texto enfatizado</em>.</p>

<br>

La etiqueta <br> inserta un salto de línea en el texto.

        <p>Línea 1<br>Línea 2</p>

<hr>

La etiqueta <hr> inserta una línea horizontal que separa el contenido.

        <hr>

<form>

La etiqueta <form> define un formulario que recoge datos del usuario y los envía a un servidor.

        <form action="/submit" method="post">
            <input type="text" name="nombre" placeholder="Tu nombre">
            <input type="submit" value="Enviar">
        </form>

<input>

La etiqueta <input> permite al usuario ingresar datos en un formulario. Puede tener varios tipos, como texto, botón, casilla de verificación, etc.

        <input type="text" name="nombre">

<textarea>

La etiqueta <textarea> crea un área de texto en la que los usuarios pueden escribir múltiples líneas de texto.

        <textarea name="comentario" rows="4" cols="50">Escribe tu comentario aquí...</textarea>

<select> y <option>

La etiqueta <select> crea un menú desplegable, y la etiqueta <option> define las opciones dentro del menú.

        <select name="opciones">
            <option value="1">Opción 1</option>
            <option value="2">Opción 2</option>
        </select>

<label>

La etiqueta <label> asocia un texto descriptivo con un control de formulario, como un campo de entrada o una casilla de verificación.

        <label for="nombre">Nombre:</label>
        <input type="text" id="nombre" name="nombre">

<button>

La etiqueta <button> crea un botón que puede realizar una acción cuando se hace clic en él.

        <button type="button">Haz clic aquí</button>

<iframe>

La etiqueta <iframe> permite incrustar otra página web dentro del documento actual.

        <iframe src="https://www.ejemplo.com" width="300" height="200"></iframe>

<b>

La etiqueta <b> hace que el texto dentro de ella aparezca en negrita, pero sin agregar énfasis semántico.

        <p>Este es un <b>texto en negrita</b>.</p>

<i>

La etiqueta <i> hace que el texto dentro de ella aparezca en cursiva, pero sin agregar énfasis semántico.

        <p>Este es un <i>texto en cursiva</i>.</p>
