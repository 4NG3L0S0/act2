5 Etiquetas Adicionales en HTML3

<blockquote>

La etiqueta <blockquote> se utiliza para indicar una cita en bloque. Generalmente se muestra con sangría.

        <blockquote>
            "Esta es una cita larga de un autor famoso."
        </blockquote>

<cite>

La etiqueta <cite> se usa para citar títulos de obras o fuentes. Se muestra en cursiva.

        <p>Este artículo fue inspirado por <cite>El Quijote</cite> de Cervantes.</p>

<code>

La etiqueta <code> se utiliza para mostrar fragmentos de código de programación.

        <p>El siguiente código muestra un mensaje en la consola:</p>
        <code>console.log('Hola, Mundo');</code>

<pre>

La etiqueta <pre> se usa para mostrar texto con formato predefinido, respetando los espacios y saltos de línea.

        <pre>
        Línea 1
        Línea 2
            Línea con sangría
        </pre>

<sup> y <sub>

Las etiquetas <sup> y <sub> se usan para mostrar texto en superíndice (arriba) o subíndice (abajo), respectivamente.

        <p>Este es un texto con superíndice: E=mc<sup>2</sup></p>
        <p>Este es un texto con subíndice: H<sub>2</sub>O</p>
